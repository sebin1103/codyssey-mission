# 9번 과제 — 카이사르 암호 해독기

화성 기지 비상 저장소의 잠금을 해제하기 위해, ZIP 파일 속 암호문을 카이사르 암호 기법으로 해독하는 프로그램입니다.

---

## 파일 구성

```
9/
├── main.py        # 카이사르 암호 해독 메인 프로그램
├── password.txt   # 해독할 암호문 (ZIP에서 추출)
└── result.txt     # 해독 완료 후 자동 생성되는 결과 파일
```

---

## 실행 방법

```bash
python3 main.py
```

`password.txt`와 `main.py`가 같은 폴더에 있어야 합니다.

---

## 동작 흐름

```
password.txt 읽기
      ↓
자리수 0~25 순서대로 카이사르 해독 시도
      ↓
매 자리수마다 결과 출력 (눈으로 확인 가능)
      ↓
사전 단어 발견? ──Yes──▶ 즉시 탐색 중단 → result.txt 저장
      │
      No
      ↓
26자리 모두 시도 후 사용자가 직접 자리수 입력 → result.txt 저장
```

---

## 함수 설명

### `read_password_file(file_path)`
`password.txt`를 열어 암호문 문자열을 반환합니다.
`FileNotFoundError`, `PermissionError`, 그 외 예외까지 3단계로 처리하며, 오류 발생 시 `None`을 반환합니다.

### `decode_with_shift(target_text, shift)`
카이사르 암호의 핵심 해독 로직입니다. 지정한 자리수만큼 알파벳을 역방향으로 이동시킵니다.

- 소문자(a-z), 대문자(A-Z)는 각각 자리수만큼 이동
- 공백, 숫자, 특수문자는 변환 없이 그대로 유지
- 모듈러 연산(`% 26`)으로 알파벳 범위를 벗어나지 않게 처리

```
원리: 암호 문자의 위치 - 자리수 = 원래 문자의 위치
예시: shift=19 → 'B' (1) - 19 = -18 → -18 + 26 = 8 → 'I'
```

### `caesar_cipher_decode(target_text)`
0번부터 25번까지 26가지 자리수를 순서대로 시도하며 결과를 출력합니다.
매 자리수마다 `decode_with_shift()`를 호출하고, 결과를 화면에 출력하여 눈으로 확인할 수 있게 합니다.

**보너스 과제** — 텍스트 사전을 내장하고 있으며, 해독 결과에 사전 단어가 포함되면 즉시 반복을 중단하고 결과를 반환합니다.

```python
dictionary = [
    'mars', 'love', 'base', 'roman', 'caesar',
    'system', 'secret', 'open', 'door', 'the'
]
```

### `save_result_file(file_path, text)`
해독된 최종 결과를 `result.txt`에 저장합니다.
`PermissionError`를 포함한 예외처리가 적용되어 있습니다.

## 실행 결과 예시

```
[암호문 로드] "B ehox Ftkl"

==================================================
  카이사르 암호 해독 시작
==================================================
  자리수 [00] : B ehox Ftkl
  자리수 [01] : A dgnw Esjk
  자리수 [02] : Z cfmv Drij
  ...
  자리수 [19] : I love Mars

[자동 탐지] 키워드 "mars" 발견 → 자리수 19번에서 탐색 중단

[저장 완료] 해독된 암호가 "result.txt" 에 저장되었습니다.
[최종 결과] I love Mars
```

암호문 `"B ehox Ftkl"`은 **자리수 19**에서 `"I love Mars"`로 해독됩니다.

## 개발 환경 및 제약 조건

- Python 3.x
- 외부 라이브러리 사용 없음 (Python 기본 제공 기능만 사용)
- PEP 8 스타일 가이드 준수
- 문자열 기본 표기: `' '` 사용
- 모든 파일 입출력에 예외처리 적용
